Rename following file extensions from .txt to .properties and place them in appropriate directories:
1. ccav.txt => /fasset/src/main/resources/ccav.properties
2. ccavParams.txt => /fasset/src/main/resources/ccavParams.properties
3. conf.txt => /fasset/src/main/resources/conf.properties
4. log4j.txt => /fasset/src/main/java/log4j.properties
5. views.txt => /fasset/src/main/java/views.properties
6. messages.txt => /fasset/WebContent/WEB-INF/messages.properties
